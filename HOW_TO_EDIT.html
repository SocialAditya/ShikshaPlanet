📁 File Structure — Konsi File Kya Hai?

📄

index.html

Sabse important file. Saara text, headings, prices, descriptions, links yahan hain.

90% changes yahan
🎨

style.css

Colors, fonts, sizes, animations — website ka poora design yahan control hota hai.

Colors badlne ke liye
⚙️

script.js

Form validation, animations, navbar scroll effect. Generally ise mat chheedo.

Advanced only
🖼️

images/ folder

Sabhi 4 book cover images yahan hain. Nai image yahan copy karke add karo.

⚡ 3 Easy Steps — Koi Bhi Change Karo

1

File kholo: index.html ko Notepad++ ya VS Code mein kholo

2

Text dhundho: Ctrl + F dabao, jo text dikhta hai website par, woh type karo — seedha wahan pahunch jaoge

3

Save karo aur check karo: Ctrl + S se save karo, phir index.html double-click karo browser mein dekhne ke liye

💡 Best Tip! VS Code + "Live Server" extension install karo. Save karte hi browser automatic refresh ho jaata hai — baar baar manually refresh nahi karna padta!
⚠️ Save karna mat bhoolo! Changes karne ke baad hamesha Ctrl+S dabao. Browser mein changes dikhne ke liye Ctrl+Shift+R (hard refresh) dabao.

Changes karne ke baad browser mein nahi dikh raha? ▼
Ctrl+Shift+R dabao (Hard Refresh). Yeh browser cache clear karke fresh version load karta hai. Agar phir bhi nahi dikh raha, file save hua hai ya nahi check karo (Ctrl+S).
Book image nahi dikh rahi — error aa raha hai? ▼
3 cheezein check karo:
  1. Image images/ folder mein hai ya nahi?
  2. index.html mein src="images/NAAM.png" — exact naam match karta hai?
  3. File extension sahi hai? (.png vs .jpg vs .PNG — case sensitive!)
WhatsApp button click karne par kuch nahi hota? ▼
Phone/computer mein WhatsApp installed hona chahiye. Desktop par WhatsApp Web browser mein open hota hai — isliye internet connection chahiye. Mobile par seedha WhatsApp app open hoga.
Mobile par website ka layout bigad gaya? ▼
style.css ke @media sections (line ~580 se aage) mat chheedo — yeh responsive design hai. Agar kisi aur change se bigad gaya ho toh us specific change ko Ctrl+Z se undo karo.
Galti se kuch delete ho gaya — wapas kaise laun? ▼
Agar file still open hai: Ctrl+Z dabao (unlimited undo).

Agar file save ho gayi aur band bhi kar diya: Antigravity se bol do — original file restore kar denge!
Ek aur section (jaise FAQ ya Gallery) add karna hai? ▼
Seedha Antigravity mein message karo: "Website mein ek naya [SECTION TYPE] section add karna hai" — aur main woh section build karke file mein add kar doonga!
Website ko hosting par deploy kaise karein? ▼
Sabse easy options:
  1. Netlify Drop — app.netlify.com/drop par jaao, poora folder drag & drop karo — free hosting!
  2. GitHub Pages — GitHub account banana hoga
  3. cPanel Hosting — Agar aapke paas hosting hai toh File Manager mein upload karo
Konsa editor (software) use karoon? ▼
Recommended (free):
  • VS Code — code.visualstudio.com — Best option, "Live Server" extension ke saath automatic preview milta hai
  • Notepad++ — notepad-plus-plus.org — Simple aur fast
  • Regular Notepad — Windows built-in, works but no syntax highlighting
